h1. CheerLights

"CheerLights":http://www.cheerlights.com is an "ioBridge Labs":http://www.iobridge.com/labs project that allows people's lights all across the world to synchronize, stay linked based on social networking trends. This is a way to connect physical things with social networking experiences and spread cheer at the same time. We are all connected.
